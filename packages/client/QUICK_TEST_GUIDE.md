## 🐛 常见问题排查

### 问题 1: 折叠按钮不显示

**可能原因**:
- `onToggleCollapse` prop 未传递

**解决方案**:
```tsx
<ConversationSidebar
  // ... 其他 props
  isCollapsed={isSidebarCollapsed}
  onToggleCollapse={() => setIsSidebarCollapsed(!isSidebarCollapsed)}
/>
```

### 问题 2: 动画不流畅

**可能原因**:
- 浏览器硬件加速未启用
- CSS 属性使用了非 GPU 加速的属性

**解决方案**:
- 确保使用 `transform` 和 `opacity` 进行动画
- 检查浏览器设置

### 问题 3: 渐变不显示

**可能原因**:
- Tailwind CSS 未正确配置
- 浏览器不支持 CSS 渐变

**解决方案**:
- 检查 `tailwind.config.js`
- 更新浏览器版本

### 问题 4: backdrop-blur 无效

**可能原因**:
- 浏览器版本过旧
- 缺少 `-webkit-backdrop-filter` 前缀

**解决方案**:
- 更新浏览器到最新版本
- 添加 vendor prefixes
