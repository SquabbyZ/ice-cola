## 🔍 API 验证（打开 Network 标签）

### 请求检查
- [ ] `GET /api/extensions` - 获取扩展列表
- [ ] `POST /api/extensions/{id}/install` - 安装扩展
- [ ] `DELETE /api/extensions/{id}/uninstall` - 卸载扩展
- [ ] `POST /api/extensions/{id}/enable` - 启用扩展
- [ ] `POST /api/extensions/{id}/disable` - 禁用扩展

### 响应检查
- [ ] 所有请求返回 200 状态码
- [ ] 响应数据格式正确
- [ ] 无 404 或 500 错误
