# 用户认证模块测试用例

## TC-001: 用户注册成功
**前置条件**: 用户邮箱未被注册
**步骤**:
1. 提交注册请求 (email, password)
2. 验证返回的 user 对象

## TC-002: 用户注册失败 - 邮箱已存在
**前置条件**: 邮箱已被注册
**步骤**:
1. 提交注册请求 (已有邮箱)
2. 验证返回 409 Conflict

## TC-003: 用户登录成功
**前置条件**: 用户已注册
**步骤**:
1. 提交登录请求 (email, password)
2. 验证返回 JWT token

## TC-004: 用户登录失败 - 密码错误
**前置条件**: 用户已注册
**步骤**:
1. 提交登录请求 (错误密码)
2. 验证返回 401 Unauthorized

## TC-005: 获取当前用户信息
**前置条件**: 用户已登录
**步骤**:
1. 发送 GET /auth/me (带有效 token)
2. 验证返回的用户信息
